# To use this project 
## You must fork the project before clone it

``` 
git clone https://github.com/cloudsheger/jenkins-sonar-artifactory.git

```

# How to run docker compose 
```
docker-compose up -d
```

# How to down docker containers

```
docker-compose down
```
